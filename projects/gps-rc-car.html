GPS Autonomous RC Car

Navigating waypoints using GPS and compass

Github Documentation

Project Overview

This project transforms a old standard RC 1/16 chassis into a fully autonomous vehicle capable of navigating to specific GPS coordinates with Colin Malone. This project was made for a project based engineering course fall 2022 and was finished early 2023. Using an Matek 743 Wing flight controller as the brain, with the mission planner software to make the car navigate a waypoint based path.

How it Works

The car operates on a "Drive-to-Target" logic. It retrieves its current position via the BN-220 GPS Module and determines its current orientation using built in magnetometer in the Matek 743 Wing. By calculating the bearing between the current location and the target, the Matek adjusts the steering servo and motor speed to reach the destination.

Key Features

  • Waypoint Navigation: Input coordinates for the car to follow in sequence.
  • FPV Camera: Long Range FPV allowed visual tracking autonomous mission at long distance.
  • Failsafe Systems: Includes logic to stop the vehicle if GPS lock is lost and ability to take manual control if needed.

Hardware Stack

  • Matek 743 Wing
  • BN-220 GPS Module
  • 3DR Radio v1 (Communication with base station)
  • FlySky Receiver (Manual Override)

Development Progression

Stage 1: Base Platform

We started with a old standard 4WD RC chassis. We initially needed to repair issues with a broken transmission, deep clean wheels/suspension and making sure we got the car working well.

Stage 2: Electronics Testing

During electronic testing, we integrated the Matek 743 Wing with the BN-220 GPS Module as well as hooked up the SBUS on the receiver. We also laser cut an acrylic top plate to help old the test components. We also were able to get our first drive tests

Stage 3: Custom CAD & Mounting

To secure the electronics, we designed a custom 2 part top plate in CAD. We used a combination of laser cutting and 3D printing ot attach all the components as well as leaving extra mounting holes for future teams to work with the project.

Stage 4: Final Assembly & Field Testing

The final build features a clean wire management system and 3D-printed enclosures. When the car it on you can communication with it using the mission planner software to be able to tune setting like desired speed and pathing. This is also where we implement the manual override to be able to do course correction.
