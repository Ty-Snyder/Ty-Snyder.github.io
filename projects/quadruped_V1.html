Quadruped V1

Servo based legged robot

Project Overview

A project I started in 2023 as a fully custom quadruped project. Uses a Raspberry Pi 4B as the brain, which uses the integrated bluetooth to connect to a generic controller.

How it Works

The V1 quadruped works of a simple open loop control method which is planned to change to closed loop in V2. I use inverse kinematics along with with a time based path to send calculate the angle of each joint at the legs for any given moment. The Raspberry Pi then sends the desired position to the Pololu Mini Maestro which generates the PWM signal for the servo to go to the correct angle.

Hardware Stack

  • Raspberry Pi 4B
  • Pololu Mini Maestro 18-Channel
  • DS3218 20KG Servo
  • Zeee 7.6V 8000mAh

Sub Sections

Inverse Kinematics

While in early stages of this project I was in the works of learning the matrix inverse solution. Due to the joint system I was able to realize the simpler and more effective version through using the 2 triangles formed by the leg segments to solve for the end effector with basic trig, law of sin and law of cos rules.

Software

The software is all in Python on a raspberry Pi 4B. While the code is not super efficient it does work. I have learned a lot in the last couple years to greatly improve it and make it a closed loop system. V2 will use a Teensy 4.1 in C++ for a state machine and a raspberry pi for a higher level mission planner and basic control input.

Hardware

The entire quadruped is 3d printed by my own printer. I used primarily Autodesk Fusion for designing. While the body stayed consistent in design the legs went under multiple version to improve strength, connection with servo and more. One of the larger problems with the legs which is being worked on for version 2 is the feet due to being PLA had trouble with low friction coefficient as long as due to the feet having no squish made the bot walk on points. I plan to use TPU or a casted polyurethane to increase friction and allow squish to spread surface area slowing the tip of the bot. The new version will also focus better weight balancing.
