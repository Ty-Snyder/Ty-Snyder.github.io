Executioner

15lb Combat Robot

Project Overview

A project done in collaboration with the Placer Robotics Combat Program. Executioner was a small team project between me, Dorian Todd and Colin Malone. We used Autodesk Fusion for a full custom Cad which we spent countless hours working on and innovating from version to version. We ordered water jet aluminum frame, then had to use a drill press, 3 axis cnc and tapping equipment to get the frame assembled properly. The weapon is 4340 Steel which was EDM'ed to shape then went through heat treating to harden.

Key Features

  • Bent Titanium Front Corners: Protect from horizontal weapons.
  • Tall Ears: Allows the ability to drive upside down and allow for full spin up.
  • 4 lb Egg Beater Weapon: Egg beaters designs help optimize MOI which with a lot of weight makes the weapon store a lot of energy.

Technical Specifications

Weight Class 15lb (Dogeweight)
Weapon Type 4340 Steel Egg Beater
Weapon Speed 11,000 RPM
Drive Speed 10 ft/s
Chassis 6061-T6 Aluminum
Armor Grade 5 Titanium and UHMW
